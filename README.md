# Azure Landing Zone Terraform

This repository contains a production-ready Azure Landing Zone built with Terraform following the Azure Cloud Adoption Framework (CAF).

## Planned Features
- Hub and Spoke Architecture
- Azure Firewall
- Azure Bastion
- VPN Gateway
- Private DNS
- Azure Monitor
- Log Analytics
- Key Vault
- Storage Account
- Remote Terraform State
- GitHub Actions CI/CD
- Multi-environment (dev/test/uat/prod)
