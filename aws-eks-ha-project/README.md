# AWS Multi-Region EKS HA Terraform Project

This project provisions a highly available multi-region AWS EKS infrastructure using Terraform.

Regions:
- us-east-1
- us-west-2

Features:
- Hub and Spoke VPC architecture
- Transit Gateway
- Multi-AZ EKS clusters
- GitHub Actions CI/CD
- CloudWatch monitoring
- Prometheus and Grafana
- Terraform remote backend
